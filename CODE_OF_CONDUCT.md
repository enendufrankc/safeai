# SafeAI Code of Conduct

## Our commitment

We are committed to making participation in the SafeAI project respectful, inclusive, and harassment-free for everyone.

## Expected behavior

- Be respectful in technical and non-technical interactions.
- Provide constructive feedback focused on code, ideas, and outcomes.
- Assume good intent, ask clarifying questions, and avoid personal attacks.
- Respect different levels of experience, backgrounds, and communication styles.

## Unacceptable behavior

- Harassment, discrimination, threats, or intimidation
- Personal insults, derogatory language, or sustained hostile behavior
- Doxxing or sharing private information without consent
- Disruptive trolling, spamming, or bad-faith participation

## Scope

This Code of Conduct applies in all project spaces, including:

- GitHub issues, pull requests, and discussions
- Project communication channels and events
- Any public or private communication representing the project

## Reporting

If you experience or witness unacceptable behavior:

- Report privately to project maintainers listed in `MAINTAINERS.md`.
- If private contact details are not yet listed, use repository administrators via GitHub.
- Do not include sensitive personal details in public issues.

## Enforcement

Maintainers may take any action deemed appropriate, including:

- Warning
- Content removal
- Temporary ban
- Permanent ban

Enforcement decisions prioritize community safety and project integrity.

## Attribution

This policy is informed by standard open-source conduct practices, including the Contributor Covenant.
